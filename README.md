# nlp_study


## 개요
NLP (National Language Processing) API를 이용하기 위하여 ML(Machine Learning)를 우선적으로 학습하고, 추후 NLP를 학습한다.


## 설치(Install)
```shell
$ git clone https://github.com/JongMinLee0/nlp_study.git
```


## 라이브러리(Libaray)
```shell
$ pip install jupyter
$ pip install sklearn
```


## 실행 (Run)
```shell
$ cd nlp_study
$ jupyter notebook
```
